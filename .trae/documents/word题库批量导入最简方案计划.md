# Word题库批量导入最简方案计划

## 目标
- 在不改动后端导入协议（仍走现有 Excel 批量导入）的前提下，提供一条对教师最省事的路径：**Word 题库文本一键转 Excel 模板**。
- 优先支持你给出的题型文本格式，例如：
  - `1. 题干（ ）。A...B...C...D...【答案】B。解析：...`

## 结论（推荐方案）
- 最合适方案：新增一个“**Word/文本转Excel模板**”前端小工具页（或放到题库页弹窗内），流程为：
  1. 粘贴 Word 内容（或上传 `.docx` 提取文本）
  2. 自动解析题号、题干、选项、答案、解析、题型
  3. 生成系统现有导入模板所需字段
  4. 导出 `.xlsx`，再走现有“批量导入”接口
- 原因：实现成本最低、与现有后端完全兼容、上线风险最小、教师使用门槛最低。

## 实施步骤

### 第1步：确认导入模板字段与映射
- 梳理后端 Excel 导入 DTO 字段（content/optionA-D/type/answer/analysis/score）。
- 定义解析结果到模板字段映射规则：
  - `题干` -> `content`
  - `A/B/C/D` -> `optionA/optionB/optionC/optionD`
  - `【答案】X` -> `answer`
  - `解析：...` -> `analysis`
  - 默认 `type=SINGLE`，多选答案含`,`时设 `MULTIPLE`，判断题按关键词设 `JUDGE`
  - `score` 先给默认值（如 1 或 5，可配置）

### 第2步：实现文本解析器（先覆盖 80% 常见格式）
- 设计正则与状态机组合解析：
  - 题目切分：按 `^\d+[\.、]` 分段
  - 选项识别：`A.`/`A、`/`A ` 等多写法兼容
  - 答案识别：`【答案】B`、`答案：B`、`参考答案：B`
  - 解析识别：`解析：...`、`【解析】...`
- 容错策略：
  - 缺失字段时标记“待人工补全”
  - 识别失败题目进入“异常列表”，不阻断全量导出

### 第3步：前端工具页与交互
- 新增“Word转Excel”入口（放在题库管理页批量导入区域附近）。
- 输入方式：
  - 先做“粘贴文本”模式（最快）
  - 可选支持 `.docx` 文件读取文本
- 结果预览：
  - 表格预览解析结果
  - 高亮异常题目并允许就地编辑
- 导出按钮：
  - 一键下载 `.xlsx`（字段与后端模板完全一致）

### 第4步：联通现有导入链路
- 导出的 Excel 直接使用当前导入接口，不改后端协议。
- 在题库导入区补充操作提示：
  - “Word题库先转模板，再批量导入”
  - 提供示例格式与常见失败原因说明

### 第5步：测试与验收
- 用你给的样例 + 多组历史题库做回归：
  - 单选/多选/判断/主观题
  - 不同答案写法（`【答案】B`、`答案:B`）
  - 含年份来源信息与括号空位（`（ ）`）
- 验收标准：
  - 常见题库格式解析成功率达到可用标准（目标 >90%）
  - 导出后可被系统现有导入接口稳定接收
  - 异常题目可在导入前人工修正

## 风险与回避
- 风险：Word 源文件格式极不统一。
- 回避：
  - 首版聚焦“文本粘贴 + 正则容错”，避免直接处理复杂 Word 样式树
  - 对无法自动识别的题目提供异常清单与手动修复入口

## 分阶段落地建议（最省事）
- Phase 1（最快上线）：仅“粘贴文本 -> 解析 -> 导出Excel”
- Phase 2（体验增强）：支持 `.docx` 上传解析
- Phase 3（精度优化）：加入更多题型规则和解析策略配置
